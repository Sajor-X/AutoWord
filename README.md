## Quick Start

``` shell
$ python setup.py install
```

## Usage

Examples:

``` shell
$ autoWord -t 'Better Call Saul' -p ~/Desktop/Saul.docx
```

For more details:
``` shell
$ autoWord --help

Usage: autoWord [OPTIONS]

Options:
  -t, --title TEXT  Article title.
  -b, --body TEXT   Article body.
  -p, --path TEXT   Article path.
  --help            Show this message and exit.
```
